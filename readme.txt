Plugin finds .styl in active theme, keeps track of modified file times, and processes .styl to .css only when .styl file has been modified.

== Description ==

Plugin finds .styl in active theme and keeps track of modified file times. when file mod time has changed, plugin processes .styl -> saves results to same-styl-file-name.css. code triggered by wp_head.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
1. Make sure active theme has writeable-enough permissions to save .css files
